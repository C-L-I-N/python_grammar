3. 身份运算符：is, is not
```python
# == 比较值是否相等
# is 比较是否是同一个对象（内存地址）

a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a == b)      # True （值相等）
print(a is b)      # False （不是同一对象）
print(a is c)      # True （c 指向 a 的同一对象）

# 查看对象的内存地址
print(id(a))       # 例如：140234567890
print(id(b))       # 例如：140234567920 （不同）
print(id(c))       # 例如：140234567890 （与 a 相同）
```
